Filamp

This is a project only recommended for test purposes, where you can create and remove directories, add or remove files or directories, preview and download files, and even more for your server!

Download latest development release (v. 0.1.0) »

README.md »

Manage users

If you are administrator or owner, you can add users who are working in your project, remove them and manage the users and set their privileges as members, administrators and owners. Even, you can have a lot of owners!

Log

We implemented a log in this project, so you can see who has edited a file, uploaded a file, created or removed a directory, added new users to the project, and even more!

Safe

Filamp is designed and intended to be safe for the server and is one of the main goals of this project*. This project is designed to work against exploits, but they are still possible.

*Filamp is recommended for test purposes only, but it has security implementations. See the README for more information.

Open Source

Filamp is open source, which means that you can see the code and modify it if it is necessary. You can also fork and create pull requests to improve this project, or just submit an issue at the Github issue tracker.
